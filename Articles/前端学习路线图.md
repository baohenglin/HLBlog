## 第一阶段 Web网页基础

1. **HTML**

```
学习目标：学习 HTML 基础知识，学会使用基础的常用标签。
```

* (1)了解编译工具VSCode，搭建开发环境。
* (2)常用标签的作用及使用。常用标签包括：meta标签、div标签、h标签、a标签（定义超链接）、p标签、span标签、img标签、input标签、textarea标签。
* (3)锚点、路径问题
* (4)**表单标签form**、**表格标签table**、有序列表标签ol、无序列表标签ul、自定义列表标签dl。
* (5)音频：audio标签
* (6)视频：video标签 

2. **CSS**

```
学习目标：学习 CSS 基础知识，能够写出不同布局样式的页面。
```

* (1)CSS样式属性。比如 display属性、overflow、background、text-align、border、font-size、padding、margin等。
* (2)各类选择器。包括**clss选择器**、**id选择器**、**通用选择器**（Universal selector）、**属性选择器**（Attribute selectors）、**伪选择器**（pseudo-selectors，包括**伪类选择器**(pseudo-classes)和**伪元素选择器**(pseudo-elements)）、**组合选择器**（Combinators）。
* (3)**CSS样式权重**
* (4)CSS盒模型 
* (5)样式布局技术：流式布局、**float布局**、**position布局**（absolute、relative）、**FlexBox布局**、**grid布局**、**muti-column布局**（多列布局）、table布局（几乎不再使用）。
* (6)CSS特效：CSS3滤镜、CSS3自定义动画、CSS3动画合成混搭、CSS3景深透视、3D三维体构建、CSS3炫酷三维效果。
* (7)移动端Web页面开发与适配。涉及rem单位的使用、媒体查询以及移动端样式的主流适配方案。
* (8)CSS3浏览器兼容性适配。 
* (9)CSS预处理器：Less、Sass。
* (10)CSS编码规范。

实践：可以实现常见特效，例如：tab、导航、页面滚动、轮播图、弹出层、手风琴菜单、瀑布流布局等

3. **JavaScript**

* (1) 常量（const）与变量（var、let）。
* (2)**JavaScript数据类型**。共8种。分别是Number、String、Boolean、Null、undefined、Object(Data、function、Array)、Symbol(ES6)、bigInt(谷歌67)。其中Number、String、Boolean、Null、undefined属于基本数据类型。Object（Data、function、Array）属于引用类型。
* (3)各种运算符。比如三元运算符。
* (4)流程控制语句。包括if语句、switch语句、for循环语句、while循环语句等。
* (5)函数。函数的概念、函数的参数及返回值、函数的递归等。
* (6)原型链和this关键字原理
* (7) DOM操作。获取元素、添加元素、修改属性等。
* (8)各种事件。鼠标事件、点击事件、滚动事件、form事件、页面生命周期事件、事件链。
* (9)JavaScript主流浏览器兼容性适配。
* (10)编码规范。
* (11)常见算法。排序算法 

实践：实现基础算法；实现定时器；
