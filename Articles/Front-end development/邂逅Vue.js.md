### Vue 简介

Vue 是一套用于构建用户界面的渐进式框架。那么如何理解“渐进式”呢？

* 渐进式意味着可以将 Vue 作为应用的一部分嵌入其中，以带来更丰富的交互体验；
* 如果希望将更多的业务逻辑使用 Vue 实现，那么可以将 Vue 的核心库以及其生态系统(Core + Vue-router + Vuex)集成到项目中以满足各种各样的需求。

### Vue 特点

* (1)解耦视图和数据
* (2)可复用的组件
* (3)前端路由技术
* (4)状态管理
* (5)虚拟 DOM
