## Vue3.0 概览

### Vue3.0 相比 Vue2.X 的变化 

以下变化可以从 [Vue RFCs](https://github.com/vuejs/rfcs/tree/master/active-rfcs) 中查阅。

* (1)Slot API 的变化
* (2)全局 API 使用规范变化
* (3)增加了 teleport 的 API
* (4)Composition API
* (5)v-model 的变化
* (6)指令API的变化

### 如何学习 TypeScript

* (1)任何变量都声明类型；
* (2)不到万不得已不要使用any；
* (3)对对象声明接口。
