## 1.1 从 Hello World 说起

简单的事物背后往往又蕴含着复杂的机制，如果我们深入思考一个简单的“Hello World”程序，就会发现很多问题看似很简单，但实际上我们并没有一个非常清晰的思路；或者在我们脑海里有着模糊的印象，但真正到某些细节的时候可能又模糊不清了。比如对于C语言编写的Hello World程序：

```
#include <stdio.h>
int main()
{
  printf("Hello world\n");
  retrun 0;
}
```
